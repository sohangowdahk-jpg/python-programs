 10. Using map() to Calculate Length of Strings
# Program to calculate length of strings using map()

words = ["Python", "Java", "C", "JavaScript"]

lengths = list(map(lambda x: len(x), words))

print("Words:", words)
print("Lengths:", lengths)
Output
Words: ['Python', 'Java', 'C', 'JavaScript']
Lengths: [6, 4, 1, 10]
